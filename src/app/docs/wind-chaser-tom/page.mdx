# 追风汤姆操作技巧

1. 追风汤姆手上拿道具再抓起老鼠后，在飞行状态下用武器技能丢下老鼠，可以使道具留在手上并能正常使用;
2. 追风汤姆在飞行状态下用武器技能丢下老鼠后接主动技能和爪刀可以打死丢下来的老鼠，进行战术鞭尸，若底下有老鼠也可以打出双倒；
3. 追风汤姆可以利用飞行时的碰撞体积“顶”救援位来守火箭；
4. 在墙缝期，追风汤姆的武器技能可以吹走果盘，减慢老鼠的砸墙速度；
5. 追风汤姆的武器技能会被叉子挡住，可以在火箭旁卡一个叉子，再使用武器技能，让风与火箭重合，阻止老鼠救援；
6. 主动技能蓄力时点击道具键，可以存蓄力冲，期间武器技能、爪刀可以正常使用，松开主动技能键以释放蓄力冲。
