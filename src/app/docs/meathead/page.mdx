# 米特操作技巧

#### 胡椒粉技巧

以下胡椒粉简称为粉，爆炸简称为烟：

1. 烟的第一段伤害存在延迟，意味着鼠能够通过立即回血或长时间免虚弱和无敌免死
2. 粉可以被老鼠接住，无法形成烟；受部分击退且掉落鼠，则鼠原地下来，粉被撞飞，无法形成烟造成虚弱；跳在空中时，若受到掉落鼠的攻击，粉下落比鼠慢，形成烟的时间可以让鼠逃跑
3. 二级粉放下鼠，老鼠2秒内(此时间为眩晕时间)不会禁技能，但是2秒后禁技能，时间约等于猛攻，且附带高额减速
4. 手上有一个粉，且2级粉CD好，则可以原地放粉，将手上的鼠放下来，转头攻击其他离得近的鼠，容易打出双倒
5. 粉放下鼠，由于烟第一段伤害有延迟，可以打出爪刀，用于刷经验和三被动回血，挂二被动的禁疗
6. 有3被和2级粉，此时剩两人，抓住一只可以一直刷虚弱，稳赢
7. 二级粉放下鼠，若鼠有护盾，则不会眩晕(即使脚下有粉也不能)，可以秒按治疗回血
8. 开局向洞口丢烟放夹，破机器鼠
9. 手中鼠如果被天使上复活，用粉放下来，重新抓起可以刷掉

#### 被动技巧

1. 被动每层4点额外伤害，6层秒莱恩，7层秒泰菲，且被动是多段伤害
2. 七层手中无鼠修火箭可触发霸体，利用霸体来追人、守奶酪等等
3. 遇见对手无拆火箭，莱恩，仙女鼠时，可以在秒飞火箭附近的火箭刷霸体，或买一个火箭来刷，用来穿越火线
4. 绑火箭前，如果周围无老鼠，可以用2级粉放下来省一个霸体
5. 二被动禁疗无野性可触发
6. 有被动时击倒有铁血的鼠，若携带击晕，则铁血无法解除控制，配合拍子防换人以及拉远
7. 有3级被动和10层野性7乘胜，建议抓拍抓125血鼠，既可以上禁疗，回血，又可以无视铁血
8. 有一层被动可秒杀袋子
9. 多打出鼠的铁血，刷虚弱特性

#### 饭盒陷阱技巧

1. 爆炸是aoe伤害，受攻击力加成，共有两段，一段高伤害，一段低伤害有控制，若老鼠有护盾，则仅生效低伤和控制，若老鼠有固定减伤，则控制不触发，只有伤害和视野
2. 饭盒可以被道具打炸，能够炸飞海盗的火药桶
3. 饭盒有多个叠加在一起时，可以增加高度防跳救
4. 饭盒命中可以减50%推速
5. 在米可莉莉的换位防夹子跳救时，如果有饭盒在火箭下，由于队友救下立刻触发饭盒，因为有 aoe 伤害，所以什么东西都点不出来，直接被炸死
6. 主动砸爆2级饭盒可以回血
