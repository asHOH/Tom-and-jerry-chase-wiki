# 道具

**道具**是在游戏中四处散落的各式各样的一类物体。相比**地图组件**来说，道具通常展现出更强的互动性（例如可被拾取、投掷、使用、交互，可被击中、推动、移动，可被部分技能改变状态等）。道具是游戏进程中的重要组成部分。  
您可以在[道具](/items)界面查看已收录的道具信息。  
许多[衍生物](/entities)也具备道具的特性，可以看作是道具。

## 道具的分型

游戏中的道具并没有明确的分型。此处基于道具的特征，将道具大致分为三类：

- **投掷类**：可被拾取和投掷。
- **手持类**：可被拾取，点击使用。
- **物件类**：不可拾取，可通过交互等方式使用。

道具的分类影响了道具的使用和激活方法，详见后文。  
部分道具的特性复杂，并不能被准确归为上述某类。

## 拾取

只有**角色**可以拾取道具。道具被拾取后会储存在道具键上（[饮料](/itemGroups/饮料)除外，会储存在专用的饮料键上）。角色可通过将道具拖动到丢弃键上丢弃道具。  
有关角色拾取道具的部分相关内容，可查阅[角色](/mechanics/character)机制界面。

## 投掷

角色拾取“投掷类”道具后，可通过拖动道具键瞄准，松开后将道具掷出。

- 只有**静止**的投掷物才能被拾取，在运动中的道具无法拾取（除非它在某一时刻的速度恰好为0）。
- 未被投掷过的道具不会命中其它目标；投掷物被投出后，会进入**被投掷状态**，此时才能命中目标。
- 大部分投掷物都属于**易碎道具**，在命中目标（包括角色、其它阻挡物等）后会碎裂消失，因此通常情况下它们一旦被投掷就无法再捡回。少部分投掷物属于**可复用道具**（如**高尔夫球**），在静止后会退出被投掷状态。
- 在极少数情况下，**易碎道具**进入被投掷状态后，仍能因被部分道具“**卡位**”等原因而暂时速度为0，此时它们仍然保持被投掷状态，但角色接触它们时**不会**被命中，且可将其拾取，会保留被投掷状态，此时若丢弃该投掷物，则其接触地面时也会像被投掷命中一样碎裂。
- 高尔夫球、狗骨头等可复用道具在“恰好”被活塞等移动平台托举而上下移动时，可能会因此一直保持被投掷状态，命中任何接触它们的敌方目标。
- 部分具有推动或投掷效果的技能也可以使道具进入被投掷状态（如同被对应角色投掷）。
- 每个道具都有自己固定的**被投掷初速度**。大多数道具的投掷初速度相同，**高尔夫球**等道具的初速度较低，**奶酪**/**电风扇**等道具的初速度更低，**魔术卡牌**等少数道具类衍生物的初速度较高。
- 部分技能效果可以临时改变角色投掷道具时道具的初速度。人工智能角色有时会意外地具备该效果（表现为有时以奇快的速度诡异地将奶酪/电风扇等道具掷出）。

## 使用

角色可通过道具键使用“手持类”道具，或是通过道具/交互键使用部分“物件类”道具。 触发其各自效果。  
有关角色使用道具的部分相关内容，可查阅[角色](/mechanics/character)机制界面。

## 附：道具激活状态

（以下信息由玩家实验总结，部分机制尚未完全探明）  
所有道具均存在隐性的“**道具激活状态**”，会被**最后一个**与该道具互动的**角色**激活，继承其“**朝向**”。当道具被[冰面](/entities/冰面)（包括[水潭](/fixtures/水潭)和[宴会桌](/fixtures/宴会桌)）影响时，其运动方向会由激活自身的角色的朝向决定。

- 未被激活的道具默认向**右侧**滑动，但若接触冰面时的横向速度不为0，则改为向运动方向滑动。
- 已被激活的物体若在接触冰面时的横向速度较高，也可能改为向运动方向滑动。
- [机器鼠](/entities/机器鼠)也可以激活物体，并且覆盖前一个角色的激活；离开机器鼠后，会判定机器鼠对物体的激活消失，物体重新变成未激活状态；角色激活物体后进入机器鼠，角色没有消失而且保持着进入机器鼠的最后朝向，但进入机器鼠后激活不起效，出机器鼠后会恢复。
- 运动方向由物体与冰面接触的瞬间决定，角色在接触前转向不影响判定。
- 大/小纸盒，空投包裹，面粉袋，叉子，高尔夫球不受冰面影响，不在此处探讨。

部分已知的道具激活方式如下：

| **物体**     | **可被谁激活** | **激活方式**   |
| ------------ | -------------- | -------------- |
| 投掷物、饮料 | 猫&鼠&机械鼠   | 拿起           |
| 冰桶、鞭炮堆 | 猫&鼠&机械鼠   | 使用           |
| 奶酪         | 鼠&机械鼠      | 拿起           |
| 果盘         | 猫&鼠          | 发射果子       |
| 老鼠夹       | 猫&机械鼠      | 拿起           |
| 打开的老鼠夹 | 猫&鼠          | 踩中           |
| （金）修理锤 | 猫&机械鼠      | 拿起           |
| 拳头盒子     | 猫&鼠          | 猫拿起；鼠推动 |
| 风扇         | 猫&鼠          | 拿起           |
