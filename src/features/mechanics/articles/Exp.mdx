# 经验

**经验**是单次对局中用于提升角色等级的局内货币，该货币属于局内货币，不会继承至下一场对局。角色通过等级提升可以获得技能点，用于解锁技能或者强化已有技能。

## 角色等级

对局中，角色最低等级为1，最高等级为10。升级都需要消耗不同量的经验（见下表）。

| 升级 | 消耗经验 |
| ---- | -------- |
| 1→2  | 1000     |
| 2→3  | 1400     |
| 3→4  | 1800     |
| 4→5  | 2200     |
| 5→6  | 2600     |
| 6→7  | 3000     |
| 7→8  | 3400     |
| 8→9  | 3800     |
| 9→10 | 4200     |

> 例如：侦查期改版前携带知识渊博开局能完成1→2，现版本猫咪开局3级拾取第一块经验蛋糕恰好3→4，现版本猫咪4级后连续拾取第二第三经验蛋糕恰好4→5。

每升一级可获得一技能点，可对一个技能升级。技能等级上限需要通过升级提升，如角色4级解锁2级技能，角色6级解锁3级技能（兔子大表哥例外）。

## 各模式开局等级

- 侦察期改版前经典模式：双方开局均1级
- 侦察期改版后经典模式：双方开局均3级
- 5V5：双方开局均1级
- 黄金钥匙：双方开局均1级
- 疯狂奶酪：猫开局10级，鼠开局5级
- 烟花大作战：猫开局5级，鼠开局3级
- 特工行动：猫开局10级，鼠开局3级
- 沙滩排球：不开放等级
- 奔跑吧老鼠：开局1级
- 克隆大作战：双方开局均1级
- 圣诞树：双方开局均3级
- 多元乱斗：双方开局均4级

## 经验均分制

鼠方玩家是一个整体，所有鼠所获得的经验最终均会平分至每一个仍未被淘汰的鼠玩家身上（tips：火箭倒计时结束升空时即刻被判定为被淘汰）。

例如：第一块经验蛋糕会给予老鼠6000点经验值，在被拾取后6000经验将会分为平均的4份，每只老鼠都能分到1500经验。倘若鼠方出现减员，则获取经验总量不变，均分份数减少使得单份经验量提升，例如奶酪期第二分钟时鼠方总经验增长速度为28/s，在减员一人的情况下单只鼠分得9.33/s。

## 经验获取方式

### 自然增长

对局中，随着时间的推移双方角色会持续获得经验，经验自然增长速度会越来越快，直至上限。经验自然增长速度的影响因素有：

- 游戏阶段：不同时期，经验增长速度基本不同。增速：侦查期 ≤ 奶酪期 ≤ 墙缝期
- 奶酪期时，经验增长速度越来越快
- 知识渊博：唯一可以改变经验增长速度的金卡

各阶段经验自然增长速度（猫）如下：

> **侦查期**：6/s  
> **奶酪期**：6/s，每隔1min获得+1/s，上限为15/s  
> **墙缝期**：15/s  
> **备注**：若提前进入墙缝期，经验自然增长速度将直接提升至15/s。  
> **注意**：由于老鼠经验采用“均分制”，老鼠总增长速度为正常猫的4倍（如侦查期鼠总增速：24/s）。

### 基础行为

此类行为为猫方/鼠方所通用的获取经验的行为，如推奶酪、救队友、治疗等。

#### 猫方基础行为（流程类）

- 爪刀命中：+200
- 放飞老鼠：+800
- 击倒老鼠：+200
- 打机械鼠：+1000
- 抓住老鼠：+200
- 虚弱补偿：+82
- 绑上火箭：+200
- （旧版本）经验蛋糕：1800 / 1200 / 1000 / 800 / 600

#### 猫方基础行为（道具类）

- 拍子命中：+200
- 球和叉子：+200
- 触发夹子：+400
- 手枪子弹：+100
- 杯盘碗碟：+200
- 番茄：+50
- 冰块鞭炮：+100
- 其他道具：+0

#### 鼠方基础行为（流程类）

- 推入奶酪（前三块）：+2400（/人）
- 推入奶酪（后两块）：+4800（/人）
- 火箭救援：+1200
- 治疗队友：+100
- （旧版本）经验蛋糕：6000 / 4800 / 4000 / 2400 / 1600

### 特殊行为

此类行为为角色/知识卡所特有的获取经验的能力。

#### 角色特殊行为

- **海盗-火药桶**：摧毁火箭 +100
- **莱特宁-穷追不舍**：击倒目标 +400
- **魔术师-魔术戏法**：卡牌命中 +400×2
- **凯特-骄傲的学霸**：弱点击破 +100（/层）
- **佩克斯-西部老登**：依次 +800 / 1000 / 1200 / 1400 / 1600（1600往后不再增加）

#### 知识卡特殊行为

- **知识渊博（6点）**：开局 +1000；经验自然增长速度提升 40% / 70% / 100%
- **祝愿（4点）**：被放飞 +2500×3（削弱前为 +4500×3）
- **反侦察（3点）**：打机械鼠：400 / 500 / 600

## 附：策略分析

由奶酪加经验机制可得需要有尽量多的人去摸一下奶酪，所以对局中一定要养成压奶酪和摸奶酪的好习惯（当然视情况而定，比如开局节奏崩了就不能压奶酪，而是推两块保等级，若自身安全且小地图中队友有向洞口移动的倾向，一定要给队友留一口）。

等级是很重要的，比如队伍里面有尼宝，4级稳救，开局至少三人推就能直接到达4级，若开局双推每只鼠均分经验：2400\*2/4=1200，而3级升到4级需要1800经验，也就是每人还差600经验，鼠经验自然增长速度均分后为每人6/s。在第一分钟获得经验为每人360，第二分钟为7/s此时有（600-360）/7≈35（s），得出结论耗时为95s，即尼宝在开局双推且没有推入第二块奶酪的情况下95s后才有稳救能力，将会大大降低队伍胜算。

接着来对比知识渊博与反侦察这两张知识卡。在上一次侦查期大改版后，反侦察这张知识卡由情报卡大改为经验提供卡，其亲民的知识量消耗、较低的获取与升级难度以及强力的经验获取效果，被玩家们称为“小渊博”。对比见下：

### 纯经验量对比

| 项目       | 知识渊博 Lv.3                                           | 反侦察 Lv.3                                 |
| ---------- | ------------------------------------------------------- | ------------------------------------------- |
| 开局       | 立刻增长1000经验                                        | -                                           |
| 后续       | 经验增长速度稳定提升                                    | 是否起效取决于开局3min16s后你能否破坏机械鼠 |
| 最大经验量 | 理想情况下最大提供经验量为11050（但实战通常会严重溢出） | 最大提升经验量为2400                        |

- 备注：从3级升7级需要消耗9600经验

评价：各有优势。缺知识量就带反侦察，不缺可以考虑带渊博。

### 其他要素对比

| 要素       | 知识渊博 | 反侦察               |
| ---------- | -------- | -------------------- |
| 抽取难度   | 难       | 易                   |
| 升级速度   | 慢       | 快                   |
| 消耗知识量 | 多       | 少                   |
| 经验稳定性 | 稳定提升 | 不稳定（依赖机械鼠） |
| 经验量     | 大       | 中等                 |
