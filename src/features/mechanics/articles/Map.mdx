# 地图

**地图**是游戏进行的载体。本文将介绍部分有关地图的概念。

## 房间

**房间**是根据地图结构划分的基本区域类型，会影响部分状态效果判定。  
房间可以被进一步分为以下几种类型：

| **类型**     | **特点**                                                                         |
| ------------ | -------------------------------------------------------------------------------- |
| **常规房间** | 会正常在小地图和地图预览上显示，通常有各自的传送点（详见后文）                   |
| **准备间**   | 老鼠开局所待的房间，正常情况下不显示                                             |
| **彩蛋房**   | 地图中隐藏的彩蛋房间\*，正常情况下不显示                                         |
| **天堂**     | 仅在场上存在[库博](/characters/%E5%BA%93%E5%8D%9A)时才出现，为一个独立的特殊房间 |
| **地图外**   | 正常情况下无法到达，仅可通过bug抵达（请不要在正常对局中尝试）                    |

_\*在夏日游轮/大都会地图中存在的部分隐藏区域有时也被称作彩蛋房，但实际上并非单独房间。_

目前地图中的房间均为**矩形**或**由多个矩形子房间拼合形成的复合图形**结构，其中拼接而来的房间较为少见，且特性较正常地图有所不同。  
以下列举了部分根据“房间” 进行判定的内容，以及它们与复合房间的互动情况。

| **内容**                                                 | **简述**                                                                               | **互动情况**                                       |
| -------------------------------------------------------- | -------------------------------------------------------------------------------------- | -------------------------------------------------- |
| 鼠“害怕”                                                 | 同房间内有不处于[隐身](/buffs/隐身)或[隐匿](/buffs/隐匿)的猫时，老鼠会出现特殊待机动作 | 各个子房间单独判定                                 |
| [警戒](/characters/%E4%BE%8D%E5%8D%AB%E6%B1%A4%E5%A7%86) | 对同房间内老鼠施加debuff                                                               | 对整个拼合房间均生效                               |
| [炸药堆](/fixtures/炸药堆)                               | 对同房间不同目标各自造成伤害、控制、引线缩短、直接放飞等                               | 对整个拼合房间均生效                               |
| 传送点                                                   | 在传送界面点击房间后的传送目标点                                                       | 不同房间有所不同：各自有传送点/共用传送点/无法传送 |

## 传送点与复活点

**传送点**是在各个房间预设的传送位置，大多数常规房间各自有一个；**复活点**是地图中预设的角色复活位置，全图唯一。

在经典奶酪赛等大多数模式中，猫角色开局、猫角色使用天堂中的[天空扶梯](/entities/天空扶梯)、任意角色使用[魔镜](/entities/魔镜)的传送功能时，会展开**传送地图**，显示“本地图中所有的常规房间”，并可通过点击传送到**该房间内预设的传送点处**。如果对应房间没有传送点，则无法传送（此时点击后无效果，可继续选择其它房间点击）。部分拼合房间（例如[太空堡垒I](/maps/太空堡垒I)-太空）各区域共用一个传送点。

被天使杰瑞使用[天使祝福](/characters/%E5%A4%A9%E4%BD%BF%E6%9D%B0%E7%91%9E)祝福的队友，或持有3级[喜剧之王](/characters/%E8%A1%A8%E6%BC%94%E8%80%85%E2%80%A2%E6%9D%B0%E7%91%9E)效果的表演者杰瑞被放飞时，不会立刻被淘汰，而是会在**复活点**重生。若地图没有预设复活点，则此类技能直接失效，无法重生，通常情况下这种情形仅会见于部分特殊模式（例如[装饰树大作战](/modes/装饰树大作战)）。

## 地图组件与道具

**地图组件**（也被称作**场景物**）是地图中预设的各类内容，与地图本身高度绑定，具有各自的功能（例如承载、交通、娱乐等）。

**道具**是地图中随机刷新的各类内容，有时与地图挂钩，但通常情况下在各个地图均可见。道具是游戏内对抗内容的重要组成部分。

## 地图背景

**地图背景**是地图内的背景贴图，并无效果，仅用于渲染地图氛围。

> 事实上，地图背景与部分地图组件的界限并不严格。如果一个地图内容表现出了明显的“功能性”，那它通常会被归为组件，反之则归为背景。
