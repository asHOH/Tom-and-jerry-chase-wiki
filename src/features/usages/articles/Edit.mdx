# 网站编辑指南

（注意：该界面信息更新于2026.1.12，请注意数据时效性）  
欢迎来到**猫鼠wiki**，本界面将向您介绍如何撰写文章或提供数据。
板块信息目前仍在持续更新中，欢迎持续关注！  
猫鼠wiki交流群群号**615882730**，欢迎大家加入~

目前网站上的大多数信息均可以通过账号快速传输和更改，非常方便。

## 注册与登录账号

**注意！网站暂无法提供账号和密码找回服务，请牢记您的账号和密码！**

**双击网站主页底部“网站说明”标题**，即可打开注册与登录界面，填写用户名与密码后即可完成注册，之后亦可用同一用户名和密码进行登录。  
成功登录后，网站标题栏右上角会显示“头像”图标，这代表当前处于登录状态。（由于加载速度等原因，有时该按钮可能要延迟一段时间后才会出现，请耐心等候。也可尝试刷新界面）  
登录账号后，可以进行文章投稿和编辑记录的提交。

## 编辑模式

**注意！编辑模式下，更改记录会暂时在浏览器中保存，但仍存在潜在的数据丢失风险，建议及时提交更改，或在其它地方（例如本地文本文件）中进行备份保存！**  
**退出编辑模式时会删除所有改动记录，请及时提交！**

**双击网站主页底部“网站说明”标题**可进入编辑模式，**再次双击**即可离开。编辑模式下，网站主页按钮及标题图片右下角会显示黄色铅笔图标，以作标识。  
编辑模式下，可以更改**角色/知识卡/特技/道具/衍生物/游戏地图/游戏模式/状态效果界面**中的部分信息。改动只在本地进行，不会实时同步到云端。改动后会生成对应界面的改动记录。  
您可以在登录信息下拉框中选择“改动记录-提交”以提交更改，经过网站管理员审核通过后即可显示在网站上。提交时建议附加简短的更改信息（简要描述更改部分即可） 。  
_编辑模式下的角色页仍保留了“创建角色”“导出文案”等功能，这是网站建设过程中的遗留功能，现已无必要使用。_

## 文章

**注意！文章编辑器不会实时保存您的文章编辑数据，存在较大的数据丢失风险！建议使用其它文本编辑器编辑并保存后，再复制到网站的编辑器中编辑，以防数据丢失！**

登录网站后，通过**文章板块的“新建文章”按钮**可打开文章编辑页，撰写文章后即可提交，经由网站管理员审核通过后即可显示在网站上。之后如需更改文章，只需要打开对应文章，点击“编辑文章”按钮，更改后提交即可。  
文章编辑器中的“H2”“H3”“H4”用于生成子标题和文章目录。将插入光标置于子标题行处，选择对应的标题按钮即可将该行文字变为子标题。子标题会自动加粗，字号也会增大，但**请不要将子标题键当做加粗键来使用**，否则会导致文章目录极其混乱，影响浏览。  
“角色攻略”类的文章会自动附加在对应角色页的下方，方便更多人阅读。

## 其它更改

网站目前暂无法直接修改“特性”“历史记录”“卡组推荐人”等信息。如果您希望补充此类信息，欢迎通过网站主页或工具栏下方的“网站反馈”进行反馈，或加入我们的交流群后通过群内的文档进行反馈。  
若您有对网站发展的建议，亦欢迎通过这类方式进行反馈。  
_由于人力有限，通过此类方式反馈的数据可能需要较长时间后才能更新，抱歉抱歉……_
