# 网站编辑指南

（最后更新：2026.3.8）

## 注册与登录

**双击**主页底部的“**网站说明**”即可注册或登录。注册时会要求输入：

1. 用户名（登录凭证之一）
2. 昵称（公开展示的名称）
3. 密码（登录凭证之二）

此后即可用用户名和密码直接登录。  
**注意：网站不支持找回账号密码，请妥善保管！**  
登录成功后，右上角会显示一个新的头像图标（若没找到，请刷新页面），然后就能投稿文章或提交编辑记录啦。

- 文章和编辑记录提交后需经管理员审核。未登录时会以匿名形式提交编辑记录，可能导致该记录混杂在其它编辑记录中，不方便审批，建议登录后再提交编辑记录。

## 编辑模式

当前仅开放角色页的编辑模式，其余板块开发中。  
**点击角色页中名称旁的黄色铅笔**进入编辑模式。开启后，主页图标右下角显示黄色铅笔。  
编辑模式支持在线修改绝大多数界面信息（属性/特技/知识卡/技能/克制关系等），但仅在本地生效，需在下方选择“改动记录-提交”，经审核后同步至全站。建议附加简短的更改信息，便于审核。  
**编辑模式下的更改暂存于浏览器，因此清理浏览器数据或网站版本更新时可能会丢失部分数据，请及时提交或备份！**

- 更改提交时仅会提交对应界面的更改内容，其余内容仍会暂存。  
  _编辑模式的角色页仍保留了“创建角色”“导出文案”等遗留功能，现已无必要使用。_

### 附：文案的高级语法介绍（20260308版）

在部分文本区域（如角色技能介绍等）允许应用高级语法。（注意：文章暂不支持此类高级语法）  
**请务必使用英文符号。正确的语法符号在用户端会隐藏**。  
**尽量不要嵌套使用语法，因为这可能导致显示混乱**。

| 语法         | 格式                                           | 效果                                                              | 备注                                                                                                                                         |
| ------------ | ---------------------------------------------- | ----------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| 注释         | \[文本\]\(注释\)                               | 为文本添加注释                                                    | ----                                                                                                                                         |
| 跳转         | \{项目名\} 或 《项目名》                       | 生成一个站内链接，尝试跳转到对应项目                              | 支持使用项目本名或别名；存在多个匹配结果时会跳转到消歧页                                                                                     |
| 定向跳转     | \{项目名(项目类型)\} 或 《项目名\(项目类型\)》 | 生成一个站内链接，尝试优先跳转到对应项目类型的项目                | ----                                                                                                                                         |
| 可增伤害     | \{数字\}                                       | 将该数字作为基础伤害+角色基础攻击力的数值，生成一个详细注释       | 用于简便表示“该伤害受攻击力加成”。数字会被解读为基础伤害+角色基础攻击力的合计数值，注意不要填错；在非角色页使用该语法时，角色基础攻击力视作0 |
| 可增破坏伤害 | \{\_数字\}                                     | 将该数字作为基础砸墙伤害+角色基础墙缝增伤的数值，生成一个详细注释 | 用于简便表示“该破坏受墙缝增伤加成”，逻辑同上                                                                                                 |
| 粗下划线     | \*\*文本\*\*                                   | 为文本添加粗下划线                                                | ----                                                                                                                                         |
| 自定义样式   | \$文本\$tailwind样式\#                         | 为文本添加自定义tailwind样式                                      | 用于实现彩色字等高级文案内容。样式文本会作为对应的className                                                                                  |

## 文章投稿

登录后前往文章板块点击“**新建文章**”即可撰写，审核通过后即可显示。

- **H2/H3/H4**：用于生成目录。请勿将其作为加粗工具，以免目录混淆。
- **角色攻略**：将自动关联至对应角色详情页。

**注意：编辑器无自动保存功能，建议在本地编辑后粘贴。**

## 其他反馈

目前“特性”、“历史记录”、“卡组推荐人”等信息无法在线修改。可通过**交流群**或主页“网站反馈”联系我们。
