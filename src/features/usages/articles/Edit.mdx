# 网站编辑指南

（最后更新：2026.1.12）

## 注册与登录

**双击主页底部“网站说明”**即可注册或登录。注册时会要求输入：

1. 用户名（登录凭证之一）
2. 昵称（公开展示的名称）
3. 密码（登录凭证之二）  
   此后即可用用户名和密码直接登录。  
   **注意：网站不支持找回账号密码，请妥善保管！**  
   登录成功后，右上角会显示一个新的头像图标。（若没找到，请刷新页面）
   然后就能投稿文章或提交编辑记录啦。

## 编辑模式

**双击主页底部“网站说明”**进入/退出编辑模式。开启后，主页图标右下角显示黄色铅笔。  
编辑模式支持在线修改绝大多数界面信息（角色/知识卡/特技/道具/衍生物/游戏地图/游戏模式/状态效果），但仅在本地生效，需登录后选择“改动记录-提交”，经审核后同步至全站。建议附加简短的更改信息，便于审核。
**注意：编辑模式下的更改暂存于浏览器，且不完全可靠；退出编辑模式会清空记录。请及时提交或备份！**  
_编辑模式的角色页仍保留了“创建角色”“导出文案”等遗留功能，现已无必要使用。_

## 文章投稿

登录后前往文章板块点击**“新建文章”**即可撰写，审核通过后。

- **H2/H3/H4**：用于生成目录。请勿将其作为加粗工具，以免目录混淆。
- **角色攻略**：将自动关联至对应角色详情页。

**注意：编辑器无自动保存功能，建议在本地编辑后粘贴。**

## 其他反馈

目前“特性”、“历史记录”、“卡组推荐人”等信息无法在线修改。可通过**主页“网站反馈”**或交流群联系我们。
_由于人力有限，通过此类方式反馈的数据可能需要较长时间后才能更新，抱歉抱歉……_
