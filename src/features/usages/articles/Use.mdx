# 网站使用指南

（最后更新：2026.1.12）  
欢迎来到**猫鼠Wiki**，本文介绍网站现有功能。  
本板块持续更新中，欢迎关注！  
交流群：**615882730**。

## 查阅信息

可通过主页或导航栏访问：**猫角色、鼠角色、知识卡、特技、道具、衍生物、游戏地图、游戏模式、状态效果、文章、工具栏**。  
点击内容卡片可查看详情。  
**内容界面右上角可切换简洁/详细描述**。  
**关联信息标识**：

- **虚下划线**：悬停（电脑）/点击（手机）查看**注释**。
- **实下划线**：点击（电脑）/二次点击（手机）跳转**链接**。

## 搜索信息

点击导航栏右上角**放大镜图标**或按 `/` 键（电脑端）开启搜索。  
支持搜索**名称、描述、别名**等。

## 查阅文章

在**文章**板块查阅其他玩家撰写的文章，品类丰富。  
“角色教学”类文章会同步显示在对应角色页下方。

## 切换分站

主页底部可切换分站。

## 数据准确性说明

受限于以下因素，网站信息**不保证完全准确**：

1. **测量误差**：多数数据来自玩家手动测试。
2. **数据缺失**：部分数据难以测定，暂未录入。
3. **主观差异**：角色定位、推荐配置等内容带有作者主观见解。
4. **衍生概念**：为便于理解，我们定义了“衍生物”、“定位分类”等非官方数据，具有主观性。  
   我们会持续更新数据、提高其准确性，但这是个漫长的过程。

如需提供数据、修正错误或提出建议，请参考“网站编辑指南”。提前感谢支持！
